## 🚀 使用指南

### 首次使用（推荐）

```bash
# 1. 初始化项目
make setup

# 2. 激活虚拟环境
source .venv/bin/activate

# 3. 查看学习路径
cat LEARNING_PATH.md

# 4. 开始学习
make learn LEVEL=01

# 5. 查看进度
make progress
```

### 日常学习

```bash
# 查看今日建议
python progress.py --show

# 学习指定阶段
python learn.py --level 02

# 获取提示
python learn.py --hint --question B1

# 查看统计
python progress.py --stats
```
