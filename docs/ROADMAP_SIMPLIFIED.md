## 🚀 快速开始

### 开发环境准备

```bash
# 1. 克隆项目
git clone https://github.com/your-repo/pythonLearn.git
cd pythonLearn

# 2. 安装Python依赖
pip install -r requirements.txt

# 3. 初始化数据库
python init_db.py

# 4. 启动后端
python app.py

# 5. 启动前端（另一个终端）
cd web-frontend
npm install
npm run dev
```

### 生产环境部署

```bash
# 1. 安装Docker
curl -fsSL https://get.docker.com | sh

# 2. 一键启动
./start.sh

# 3. 访问应用
# http://localhost
```
