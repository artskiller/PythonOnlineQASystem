## 🚀 使用方式

### 方式1：本地运行（推荐）

```bash
# 1. 安装依赖
pip install -r requirements-web.txt

# 2. 启动服务
python web_app.py

# 3. 访问浏览器
# http://localhost:8080
```

### 方式2：Makefile快捷命令

```bash
# 一键启动（自动安装依赖）
make web

# 仅安装依赖
make web-install

# Docker部署
make web-docker
```

### 方式3：Docker部署

```bash
# 使用docker-compose
docker-compose up -d

# 或手动构建
docker build -t python-learning .
docker run -p 8080:8080 python-learning
```
