# 🌐 Python 交互式学习平台 - 使用指南

## 📖 简介

这是一个**基于Web的交互式Python学习平台**，提供了友好的图形界面，让您可以在浏览器中直接编写、运行和测试Python代码。

### ✨ 核心特性

- 🎨 **现代化UI** - 深色主题，护眼舒适
- 💻 **在线编辑器** - 基于CodeMirror，支持语法高亮、自动补全
- ⚡ **实时执行** - 即时运行代码，查看测试结果
- 📊 **进度追踪** - 可视化学习进度
- 🎯 **分类筛选** - 按难度、主题快速查找题目
- 💡 **智能提示** - 提供学习建议和答案参考
- 📱 **响应式设计** - 支持桌面和移动设备

## 🚀 快速开始

### 方式1：本地运行（推荐）

#### 1. 安装依赖

```bash
# 安装Web应用依赖
pip install -r requirements-web.txt
```

#### 2. 启动服务

```bash
# 启动Web服务器
python web_app.py
```

#### 3. 访问应用

打开浏览器访问：**http://localhost:5000**

---

### 方式2：Docker部署

#### 1. 构建镜像

```bash
docker build -t python-learning-platform .
```

#### 2. 运行容器

```bash
docker run -p 5000:5000 python-learning-platform
```

或使用 docker-compose：

```bash
docker-compose up -d
```

#### 3. 访问应用

打开浏览器访问：**http://localhost:5000**

## 📚 使用教程

### 1. 选择题目

- 在左侧边栏浏览题目列表
- 使用筛选器按**分类**或**难度**过滤
- 点击题目卡片加载题目

### 2. 编写代码

- 在代码编辑器中填写函数实现
- 支持语法高亮和自动缩进
- 使用 `Ctrl+S` (Mac: `Cmd+S`) 保存（浏览器功能）

### 3. 运行测试

- 点击 **"▶️ 运行测试"** 按钮
- 查看测试结果和输出
- 根据错误信息调试代码

### 4. 获取帮助

- **💡 查看提示** - 获取解题思路
- **📖 查看答案** - 查看完整答案代码
- **🔄 重置代码** - 恢复到初始状态

## 🔧 配置说明

### 环境变量

```bash
# Flask环境（development/production）
export FLASK_ENV=development

# 服务端口
export FLASK_PORT=5000

# 主机地址
export FLASK_HOST=0.0.0.0
```

### 安全配置

**重要**：生产环境部署时，请注意：

1. **代码执行安全** - 当前使用临时文件执行，建议使用Docker容器隔离
2. **超时限制** - 默认30秒，可在 `web_app.py` 中调整
3. **CORS配置** - 生产环境建议限制允许的域名
4. **文件权限** - 确保题目文件只读

## 🐛 常见问题

### Q1: 无法启动服务？

**A**: 检查依赖是否安装完整：
```bash
pip install -r requirements-web.txt
```

### Q2: 代码执行超时？

**A**: 检查代码是否有死循环或耗时操作。超时限制为30秒。

### Q3: 题目列表为空？

**A**: 确保 `interview_exercises/` 目录存在且包含题目文件。

### Q4: 无法查看答案？

**A**: 部分题目可能没有答案文件（`set_X_answers.py`）。

### Q5: 端口被占用？

**A**: 修改端口：
```bash
# 在 web_app.py 最后一行修改
app.run(debug=True, host='0.0.0.0', port=8080)  # 改为8080
```
