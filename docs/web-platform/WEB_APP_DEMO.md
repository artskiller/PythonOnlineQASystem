# 🌐 Web学习平台演示

## 🎉 欢迎使用Python交互式学习平台！

这是一个**全新的Web学习模式**，让您可以在浏览器中直接学习Python，无需配置复杂的开发环境。

---

## ⚡ 一分钟快速体验

### 1. 启动服务

```bash
# 方式1：使用Makefile（推荐）
make web

# 方式2：直接运行
pip install flask flask-cors
python web_app.py
```

### 2. 打开浏览器

访问：**http://localhost:8080**

### 3. 开始学习

1. 从左侧选择一个题目（例如：ML1 - 机器学习基础）
2. 在代码编辑器中填写代码
3. 点击"▶️ 运行测试"
4. 查看测试结果

## ✨ 核心功能

### 1. 在线代码编辑器

- **语法高亮** - Python代码自动着色
- **自动缩进** - 符合PEP 8规范
- **行号显示** - 方便定位代码
- **括号匹配** - 自动高亮配对括号
- **深色主题** - Monokai主题，护眼舒适

### 2. 实时代码执行

- **即时运行** - 点击按钮立即执行
- **安全沙箱** - 隔离执行环境
- **超时保护** - 30秒自动终止
- **完整输出** - 显示print()和错误信息

### 3. 智能学习辅助

- **💡 查看提示** - 获取解题思路和建议
- **📖 查看答案** - 查看完整答案代码
- **🔄 重置代码** - 恢复到初始状态
- **📊 测试结果** - 自动运行测试用例

### 4. 题目管理

- **分类筛选** - 按AI/基础/数据/算法等分类
- **难度筛选** - 按⭐简单/⭐⭐中等/⭐⭐⭐困难
- **实时搜索** - 快速找到目标题目
- **进度追踪** - 记录学习进度（开发中）

## 🔧 高级功能

### 自定义端口

```bash
# 使用环境变量指定端口
PORT=3000 python web_app.py

# 访问 http://localhost:3000
```

### Docker部署

```bash
# 构建镜像
docker build -t python-learning .

# 运行容器
docker run -p 8080:8080 -e PORT=8080 python-learning

# 或使用docker-compose
docker-compose up -d
```

### 生产环境部署

```bash
# 安装生产服务器
pip install gunicorn

# 启动（4个worker进程）
gunicorn -w 4 -b 0.0.0.0:8080 web_app:app
```

## 🐛 故障排除

### 问题1：无法启动服务

```bash
# 检查依赖
pip list | grep -i flask

# 重新安装
pip install -r requirements-web.txt
```

### 问题2：端口被占用

```bash
# 使用其他端口
PORT=3000 python web_app.py
```

### 问题3：代码执行失败

- 检查代码语法
- 查看错误信息
- 确保依赖已安装（如jieba、scikit-learn）
